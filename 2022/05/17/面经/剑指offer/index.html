链表

1 从尾到头打印链表

描述

输入一个链表的头节点，按链表从尾到头的顺序返回每个节点的值（用数组返回）。

如输入{1,2,3}的链表如下图:

返回一个数组为[3,2,1]

0 <= 链表长度 <= 10000

示例1

输入：{1,2,3}
返回值：[3,2,1]

示例2

输入：{67,0,24,58}
返回值：[58,24,0,67]

递归

function printListFromTailToHead(head)
{
    // write code here
    let result = [];
    const traverse = (node) => {
        if (node === null) {
            return null;
        }
        traverse(node.next);
        result.push(node.val);
    }
    
    traverse(head);
    return result;
}

function ReverseList(pHead)
{
    // write code here
    if (pHead === null) {
        return null;
    }
    
    let pre = null;
    let cur = pHead;
    let temp = null;
    
    while (cur !== null) {
        // 交换cur和pre的指向
        temp = cur.next;
        cur.next = pre;
        // 双指针移动
        pre = cur;
        cur = temp;
    }
    return pre;
}
module.exports = {
    ReverseList : ReverseList
};

（2）迭代

function ReverseList(pHead)
{
    // write code here
    const traverse = (pre, cur) => {
        if (cur === null) {
            return pre;
        }
        
        let temp = cur.next;
        cur.next = pre;
        return traverse(cur, temp);
    }
    
    return traverse(null, pHead);
}
module.exports = {
    ReverseList : ReverseList
};

JZ8 二叉树的下一个结点

牛客链接

描述：

给定一个二叉树其中的一个结点，请找出中序遍历顺序的下一个结点并且返回。注意，树中的结点不仅包含左右子结点，同时包含指向父结点的next指针。下图为一棵有9个节点的二叉树。树中从父节点指向子节点的指针用实线表示，从子节点指向父节点的用虚线表示。

数据范围：节点数满足$1 \le n \le 50$，节点上的值满足 $1 \le val \le 100 $

要求：空间复杂度 O(1) ，时间复杂度 O(n)

输入描述：

输入分为2段，第一段是整体的二叉树，第二段是给定二叉树节点的值，后台会将这2个参数组装为一个二叉树局部的子树传入到函数GetNext里面，用户得到的输入只有一个子树根节点

返回值描述：

返回传入的子树根节点的下一个节点，后台会打印输出这个节点

示例1：

输入：{8,6,10,5,7,9,11},8
返回值：9

示例2：

输入：{5},5
返回值："null"
说明：不存在，后台打印"null"

思路一：

找到根节点，中序遍历时记录每个节点，然后循环匹配传入的节点，则中序遍历中下一个节点就是要找的节点。

时间复杂度：O(N)，空间复杂度：O(N)

思路二：

分类三种情况搜索：

1.节点的右子树不为空，找到该节点右子树最左边的节点并返回；

2.节点右子树为空，且该节点是其父节点的左节点，直接返回其父节点；

3.节点右子树为空，该节点是其父节点的右节点，则该节点一直沿着next指针往上走，直到该节点是next指针所指向节点的左子节点；

时间复杂度：O(N)，空间复杂度：O(1)。

思路一

/*function TreeLinkNode(x){
    this.val = x;
    this.left = null;
    this.right = null;
    this.next = null;
}*/
function GetNext(pNode)
{
    // write code here
    // 先找到根节点，然后中序遍历整棵树，最后匹配目标节点
    let root = pNode;
    let nodes = [];
    let result = null;
    while (root.next) {
        root = root.next;
    }
    
    const traverse = (node) => {
        if (node === null) {
            return ;
        }
        traverse(node.left);
        nodes.push(node);
        traverse(node.right);
    }
    
    traverse(root);
    
    for (let i = 0; i < nodes.length; i++) {
        if (nodes[i].val === pNode.val) {
            result = nodes[i+1];
            break;
        }
    }
    
    return result;
}
module.exports = {
    GetNext : GetNext
};

思路二

/*function TreeLinkNode(x){
    this.val = x;
    this.left = null;
    this.right = null;
    this.next = null;
}*/
function GetNext(pNode)
{
    // write code here
    // 分类查找
    // 1. 节点的右子树不为空，找到该节点右子树最左边的节点并返回
    if (pNode.right !== null) {
        let rChild = pNode.right;
        while (rChild.left !== null) {
            rChild = rChild.left;
        }
        return rChild;
    }
    
    // 2. 节点右子树为空，且该节点是其父节点的左节点，直接返回其父节点
    if (pNode.next !== null && pNode.next.left === pNode) {
        return pNode.next;
    }
    
    // 3. 节点右子树为空，该节点是其父节点的右节点，则该节点一直沿着next指针往上走，
    // 直到该节点是next指针所指向节点的左子节点
    if (pNode.next !== null && pNode.next.right === pNode) {
        let ppar = pNode.next;
        while (ppar.next !== null && ppar.next.right === ppar) {
            ppar = ppar.next;
        }
        return ppar.next;
    }
    return null;
}
module.exports = {
    GetNext : GetNext
};

求树中某条路径和为sum的路径有多少条。该题路径定义不需要从根节点开始，也不需要在叶子节点结束，但是一定是从父亲节点往下到孩子节点。

前缀和定义

一个节点的前缀和就是该节点到根之间的路径和

前缀和对于本题的作用

题目要求的是找出路径和等于给定数值的路径总数, 而:

两节点间的路径和 = 两节点的前缀和之差

关键代码：

let res = 0;
curSum += node.val;
res += prefixMap.get(curSum - target) ?  prefixMap.get(curSum - target) : 0; // 路径和为sum的路径是否存在
prefixMap.set(curSum, (prefixMap.get(curSum) ? prefixMap.get(curSum) : 0) + 1); // 存储当前的前缀和
prefixMap.set(curSum, prefixMap.get(curSum) - 1); // 恢复状态
